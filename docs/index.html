NeuCo-Bench

NeuCo-Bench is a benchmarking framework designed to evaluate how effectively compressed embeddings preserve information for downstream tasks.

Originally developed to evaluate challenge submissions for the 2025 EARTHVISION Challenge at CVPR, NeuCo-Bench is now released for local benchmarking and evaluation. Additional technical details in NeuCo-Bench Preprint.

GitHub Repo

Overview

In Earth Observation (EO), pipelines process massive volumes of image data for analytical tasks. Traditional compression methods target pixel-level reconstruction, while Foundation Model (FM) research often overlooks embedding size. NeuCo-Bench bridges this gap by enforcing strict size constraints and evaluating compressed embeddings directly on real-world EO tasks.

Originally developed for the CVPR 2025 EARTHVISION Challenge, NeuCo-Bench now provides an open, reproducible framework for local benchmarking and comparison of embedding-based methods. It includes a growing set of EO downstream tasks and welcomes community contributions of additional datasets, metrics, and compression techniques.

Key Features

  • Model-agnostic: Evaluates any fixed-size embedding (e.g. 1024-dim vectors) for fair comparison across compression and representation learning methods.
  • Task-driven: Uses linear probes across diverse EO tasks such as land-cover estimation, cloud detection, and biomass prediction.
  • Consistent metrics: Incorporates signal-to-noise scores and dynamic rank aggregation for unified benchmarking.

To run an evaluation:

  1. Download the SSL4EO-S12-downstream dataset from Hugging Face.
  2. Encode images into fixed-size embeddings and save them as CSV files.
  3. Run NeuCo-Bench locally to generate task metrics and leaderboards — see the Quickstart section on GitHub for detailed setup instructions.
